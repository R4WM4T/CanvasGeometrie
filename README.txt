Ce Fichier 'index.html' permet de dessiner, à l'aide de l'outil canvas, des polygônes "impossibles".
C'est une illusion assez connue sur les triangles, que j'ai souhaité essayer sur tous les polygones.

Dans le 1er input, vous rentrez le nombre de côtés que vous souhaitez.
Dans le 2nd input, vous rentrez le coefficient multiplicateur qui permet de changer les dimensions du polygone.
